This means that you can show positive and negative quantities have opposite directions on a number line, determine additive inverses, show absolute value of a rational number on a number line, and understand absolute value as distance.

Glossary

Additive Inverse
A number and its opposite. What you add to a number to get zero.

The additive inverse of -5 is +5 because -5 + 5 = 0


Absolute Value
The distance a number is from zero.

6 is 6 units away from zero, so the absolute value of 6 is 6.
-6 is 6 units away; from zero, so the absolute value of -6 is 6.


Zero Pair
A zero pair is a pair of numbers that when added together equal zero.

7 and -7 make a zero pair because 7 + -7 = 0.
